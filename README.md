# Ejecutar En Desarrollo

1. Clonar Repositorio
2. Ejecutar

```Shell
pnpm install
```
3. Tener Nest CLI instalado
```Shell
pnpm i -g @nestjs/cli
```
4. Levantar la base de datos (MongoDB)
```Shell
docker-compose up -d
```

## Stack Usado

*MongoDB

*NestJs
